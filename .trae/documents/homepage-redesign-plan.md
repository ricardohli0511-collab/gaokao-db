# 首页重构：双通道 + 智能推荐 计划

## 诊断

| 问题 | 现状 | 修复 |
|------|------|------|
| 考试类型太杂 | 7个考试类型平铺标签 | 分为**高考生** / **国际生**两条清晰通道 |
| 普通查询多余 | 有 /query 页面的独立表单 | 删除，统一用智能推荐 |
| 内地高校入口 | 「内地高校查询」卡片 | 删除（聚焦香港） |
| 布局遮挡 | `-mt-10` + `flex-1` 内容被 header 覆盖 | 改为 `-mt-0` 正常流布局 |
| 标题不准确 | 「全球院校录取查询」 | 改为「香港院校录取查询」 |
| 数据真实性 | 已在上次清理完成 | 806条 DSE 真实数据 |

## 实施计划

### 改动 1：重写页面结构

**文件**：`src/app/page.tsx`（完全重写）

新结构：

```
┌─────────────────────────────────────┐
│        PageHeader                   │
│  「香港院校录取查询」                │
│  内地同学申请香港本科/副学士        │
├─────────────────────────────────────┤
│  ┌─────────┐  ┌─────────┐          │
│  │ 高考生   │  │ 国际生   │ ← 两通道卡片 │
│  │ 频道     │  │ 频道     │          │
│  │ 输入分数 │  │ IB/A-Level │         │
│  │ →匹配港校│  │ →匹配港校   │        │
│  └─────────┘  └─────────┘          │
├─────────────────────────────────────┤
│        智能推荐表单（展开）          │
│  [考试类型] [学历] [分数]            │
│  [省份/年份/选科（条件显示）]        │
│  [匹配香港院校 →]                    │
├─────────────────────────────────────┤
│        香港院校库（24所）            │
├─────────────────────────────────────┤
│         Footer                      │
└─────────────────────────────────────┘
```

### 改动 2：双通道选择卡片

用两个 state：`track`（'gaokao' | 'international'）

- 点击「高考生」→ 显示高考表单（省份+年份+分数+选科）
- 点击「国际生」→ 显示国际考试选择 + 分数输入
- 默认高三学生首选高考通道

### 改动 3：删除冗余内容

- ❌ 删除 /query 页面内的「内地高校查询」入口
- ❌ 删除「DSE 考生专属通道」独立栏（DSE 归入国际生通道）
- ❌ 删除底部 3 个快速入口卡片（本科/副学士/内地 → 合并到上方两通道）

### 改动 4：修复布局遮挡

- PageHeader `pt-20 pb-14` → `pt-12 pb-8`（减小 padding）
- 主内容区 `-mt-10` → `mt-0`（移除负边距）
- `max-w-5xl` → `max-w-4xl`（更紧凑的单列布局）
- 标题从 `7xl` → `4xl`（缩小字号，信息层级更清晰）

### 改动 5：保持已有功能

- ✅ 本科/副学士切换
- ✅ 香港院校库卡片网格（按考试类型高亮）
- ✅ 显示全部/仅显示有数据 切换
- ✅ CompareDrawer
